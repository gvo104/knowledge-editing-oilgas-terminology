# EasyEdit Upstream Reference

This repository is a research project built on top of `EasyEdit`, not a generic mirror of the framework.

To keep the public repository understandable while preserving the framework context:

- the current root `README.md` documents the research project;
- a local snapshot of the earlier upstream-style root README is preserved at [README_EASYEDIT_UPSTREAM.md](README_EASYEDIT_UPSTREAM.md);
- the additional upstream steering-oriented README remains available at [../README_2.md](../README_2.md);
- the original upstream repository is available at <https://github.com/zjunlp/EasyEdit>.

Use this reference file when you need the framework context rather than the project-specific experiment workflow.
